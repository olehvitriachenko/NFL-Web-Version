# NFL Web App

NFL Web Application built with React, TypeScript, Vite, and Electron.

## Tech Stack

- React 19
- TypeScript
- Vite
- Electron
- Tailwind CSS
- Zustand (State Management)
- TanStack Router
- React Hook Form + Yup
- Axios
- Better SQLite3
- CryptoJS
- React PDF

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Electron

```bash
npm run electron
```
